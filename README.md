# 📺 Netflix Landing Page Clone

This project is a responsive **Netflix landing page clone** created using **HTML and CSS**.  
It replicates the look and feel of Netflix’s homepage for learning and practice purposes.

---

## 🚀 Features

- Netflix-style hero section
- Responsive layout using Flexbox and Media Queries
- Embedded videos inside TV frames
- FAQ section with hover effects
- Navigation bar with logo and buttons
- Footer similar to Netflix homepage

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
  - Flexbox  
  - Media Queries  
  - Positioning  
  - Hover Effects
