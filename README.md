# 📺 Netflix Landing Page Clone

This project is a **responsive Netflix landing page clone** built using **HTML5 and CSS3**.  
It visually replicates the Netflix homepage and focuses on **layout design, responsiveness, and UI styling** for learning and practice purposes.

## 🚀 Features

- 🎬 Netflix-style hero section with background image overlay
- 📱 Fully responsive layout for desktop and mobile devices
- 📺 Embedded videos displayed inside TV mockups
- ❓ Interactive FAQ section with hover effects
- 🧭 Navigation bar with Netflix logo and action buttons
- 🦶 Footer section inspired by the official Netflix website
- 🎨 Clean UI with consistent spacing and typography

---

## 🛠️ Technologies Used (Detailed)

### 🧱 Structure
- **HTML5**
  - Semantic elements for better readability
  - Well-structured layout sections
  - Accessible and SEO-friendly markup

### 🎨 Styling & Layout
- **CSS3**
  - Flexbox for layout alignment
  - Media Queries for responsive design
  - Positioning (relative & absolute)
  - Background overlays and opacity effects
  - Hover effects for interactive elements
  - Custom buttons and typography styling

- Creating responsive designs using media queries
